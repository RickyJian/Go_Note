# Go 目錄

GO程式必須放在工作區，並對應於特定專案的目錄

## src

儲存原始程式碼

> 副檔名：.go 、 .c 、 .h 、 .s 

### Go 函式庫原始程式檔案

透過 `go install` 安裝的套件

### Go 指令原始程式檔案

宣告為 main 的程式套件，可透過 go run 執行。同一套件中的所有原始程式檔案和函數原始程式檔案處於同一個程式套件中，那麼該套件中就無法執行 `go build` 和 `go install`

```go

package main 

```

### Go 測試原始程式檔案 

透過 `go test` 執行目前套件下的測試檔案

* 檔案須以 `_test.go` 結尾
* 

## pkg

編譯後產生的`.a`壓縮檔

> 副檔名：.a

## bin

編譯後產生的可執行檔
