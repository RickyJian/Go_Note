# For

`Go` 唯一迴圈

> for initialization ; conditon ; post { }


```go

for i := 0 ;  i < 10  ; i++ {

}

```

## initialization

* 選擇性的
* 簡單陳述式
* 簡短變數宣告、加值或指派陳述、或函式呼叫

## condition

* 求值的布林運算式
* true 時迴圈繼續執行

## post

迴圈執行完後所執行的陳述

### while

實作 `while`

> for condition {}

```go

for sum < 10 {

}

```

### 無限迴圈 ( infinite loop )

> for { }
